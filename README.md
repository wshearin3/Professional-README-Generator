 ## Description
This project was created to generate README files for projects using nodejs and inquirer. The application is run locally, takes user input, and generates a markdown file displaying relevant information on the project.

 ## Installation
To use the project, nodejs needs to be installed as well as inquirer 8.2.4.

 ## Usage
